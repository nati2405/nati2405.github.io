Modeling Student Debt Dynamics

A system-dynamics simulation inspired by SEIR epidemiological models

This project models how student debt evolves across multiple financial stages using a compartmental system inspired by SEIR models. Borrowers transition between states representing financial exposure, accumulation, active repayment stress, and resolution.

The simulation highlights how policy-sensitive parameters influence peak financial risk. The combined Exposed + Infected curves represent the maximum concentration of borrowers under financial strain.
